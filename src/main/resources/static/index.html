Bienvenido

Esta aplicación usa varias páginas HTML. Cada módulo está en su propia página para mantener una navegación clara y ordenada.

  • Especialidades: registrar nuevas especialidades
  • Pacientes: registrar nuevos pacientes
  • Médicos: registrar nuevos médicos
  • Turnos: crear o cancelar turnos
  • Consultas: ver listados y resultados
